GameData:

 

Der Server benötigt eine Datei Namens "GameData.dat", die im selben Ordner wie die .exe

Datei des Servers sein muss.

Diese Datei enthält, alle für das Spiel benötigten Daten.

 

Die GameData Datei kann in diesem Programm oder in dem speziellem "GameDataGenerator"

generiert werden.

 

Sie können sich die Daten die in der GameData-Datei sind ansehen indem Sie bei dem

Menüpunkt "Von GameData-Datei laden ...", der sich im Punkt Tools befindet, auswählen.

 

Warnung: Dieser Modus dient NUR zum Betrachten der Daten.

Sollten Sie diese ändern wollen, starten Sie den Mysql-Modus und exportieren Sie die Daten,

über den "GameData Creator"!
